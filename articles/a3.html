工欲善其事，必先利其器

2016-8-15
子贡问为仁。子曰：“工欲善其事，必先利其器。居是邦也，事其大夫之贤者，友其士之仁者。” ——《论语·卫灵公》

要做好一件事，工具是非常重要的。学习前端技术，自然要用到代码编辑器。我选择了Sublime Text，一款被认为“性感无比”编辑器。在使用了近两周之后，感觉还是名副其实的，启动无比迅速，语法高亮、代码提示补全、代码折叠等功能都是相当实用的。 工匠想要使他的工作做好，一定要先让工具锋利。对于Sublime Text，它还有更多强大的功能未被用到。在参加了相关课程之后，我学习到了一些快捷键和可以提高效率的技巧，记录在此。

Sublime Text快捷键

Tab 缩进
Shift + Tab 去除缩进
Ctrl + Z 撤销
Ctrl + Y 恢复撤销
Ctrl + F 查找
Ctrl +H 替换
Ctrl + V 粘贴
Ctrl + Shift + V 粘贴并自动缩进
Ctrl + Enter 插入行
Ctrl + Shift + Enter 插入前行
Ctrl + L 选择行
Ctrl + D 选择所在词
Ctrl + K --> Ctrl +D 跳过选取所在词
Ctrl + / 注释整行（如已选择内容，同“Ctrl+Shift+/”效果）
Ctrl + Shift + / 注释已选择内容
Ctrl + [ 减少缩进
Ctrl + ] 增加缩进
Ctrl + Shift + [ 折叠代码
Ctrl + Shift +] 展开代码
Ctrl + K + T 折叠属性
Ctrl + K + 0 展开所有属性
Ctrl + Shift + UP与上行互换
Ctrl + Shift + DOWN与下行互换
Ctrl + M 光标移动至括号内开始或结束的位置
Ctrl + Shift + M 选择括号内的内容（继续按选择父括号）
Ctrl + Shift + P 命令模式
Ctrl + N 新建文件
Ctrl + Shift + A 选择光标位置父标签对儿
Ctrl + Shift + D 复制当前行
Alt + F3 全选相同模式
Ctrl + Shift + L 拆分行
Ctrl + T 词互换
Ctrl + K + Backspace 从光标处删除至行首

Sublime Text技巧

产生多行游标

Ctrl + D 多次选择
Ctrl + D --> Alt + F3 全选
Ctrl + A --> Ctrl + Shift + L 拆分行
鼠标放在单词上，按Shift + 鼠标右键下拉

快捷输入

例1：输入10个div : 输入 div*10 ，然后 Tab （该快捷键修改过）
例2：输入10个带item类的li元素 : 输入 ul>.item*10 ，然后 Tab （该快捷键修改过）
例3：输入10个带item类的li元素且每个li的内容是test1、test2这样的内容 : 输入 ul>.item{test$}*10 ，然后 Tab （该快捷键修改过）
